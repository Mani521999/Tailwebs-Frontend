# Learning Management System (LMS) - Assignment Portal

A full-stack Assignment Portal where teachers can create, publish, and track assignments, and students can view and submit their work.

- **Student Workflow:** View all published assignments, track deadlines, and submit text-based answers.
- **Modern UI:** Built with React, Tailwind CSS, and Lucide icons.
- **Robust Backend:** Node.js with TypeScript, JWT Authentication, and MongoDB.
